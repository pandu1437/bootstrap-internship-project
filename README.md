## Project Overview
This project is a **modern, responsive website** built using **Bootstrap 5**.  
It includes three pages: **Home, About, and Contact**, featuring:  

- Navbar with responsive collapse  
- Hero / Carousel sections  
- Cards for services/features  
- Modal for newsletter subscription  
- Contact form with validation  

The project was **completed quickly (~1 hour)** with the guidance of **ChatGPT**.

**Live Project:** [https://pandu1437.github.io/bootstrap-internship-project/](https://pandu1437.github.io/bootstrap-internship-project/)  
**GitHub Repository:** [https://github.com/pandu1437/bootstrap-internship-project](https://github.com/pandu1437/bootstrap-internship-project)

## Step-by-Step Approach

1. **Project Setup**  
   - Created folder structure: `index.html`, `about.html`, `contact.html`, `style.css`.  
   - Linked **Bootstrap 5 CDN** for styles and scripts.  

2. **Component Selection**  
   - Using ChatGPT guidance, selected and adapted Bootstrap components:  
     - Navbar, Hero/Carousel, Cards, Modal, Contact Form  

3. **Page Design**  
   - **Home Page:** Hero carousel, features section, subscription modal.  
   - **About Page:** Services cards and hero description.  
   - **Contact Page:** Form with validation.  
   - Ensured **responsive design** for mobile, tablet, and desktop.  

4. **Deployment**  
   - Uploaded files to GitHub and deployed via **GitHub Pages**.
